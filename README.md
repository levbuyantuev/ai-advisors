# AI Советники

Мульти-агентный чат с AI-советниками. Получите советы от 5 разных ИИ-персон одновременно.

## Советники

- 📊 **Аналитик** (Gemini 2.5 Flash) — объективный анализ
- 🌟 **Оптимист** (Llama 3.3 70B) — возможности и мотивация
- 🔍 **Критик** (Mistral Small 3.1) — слабые места и риски
- ♟️ **Стратег** (Qwen 2.5 72B) — долгосрочные планы
- 🧠 **Психолог** (Llama 3.1 8B) — эмоции и мотивация

## Деплой на Vercel

1. Fork этого репозитория
2. Подключите к Vercel
3. Добавьте переменную окружения: `OPENROUTER_API_KEY` (получить на [openrouter.ai](https://openrouter.ai))
4. Деплой!

## Стек

- React + Vite + TypeScript
- Tailwind CSS + shadcn/ui
- Vercel Serverless Functions
- OpenRouter API (OpenAI-совместимый)
